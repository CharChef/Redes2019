5.	La configuración de las interfaces de los routers se encuetra en los
	archivos de configuración /etc/zebra/zebra.conf dentro del
	directorio de cada router, dichos archivos se geneneraron usando la
	la herramienta zebraen cada terminal accediendo con el comando
	"telnet localhost zebra", luego con "enable" se activo el modo root,
	posterior con el comando "config terminal", y para con figurar las
	interfaces "interface eth0". Una vez finalizada se ejecuta "write"
	para volcar las configuraciones en memoria a los archivos zebra.conf
	antes mecionados.
	Utilizando el comando "ip address show" en cada router se verifico
	la existencia de las interfaces y que las mismas esten activas
